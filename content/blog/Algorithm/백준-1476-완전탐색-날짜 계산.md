[문제](https://www.acmicpc.net/problem/1476)

```c++{3}
#include <cstdio>

int main() {
	int E, S, M;
	int e = 0, s = 0, m = 0;
	int time = 0;
	scanf("%d %d %d", &E, &S, &M);
	while (!(e == E && s == S && m == M)) {
		e = time % 15 + 1;
		s = time % 28 + 1;
		m = time % 19 + 1;
		time++;
	}
	printf("%d", time);
}
```

- E S M 이 일치할 때까지 완전탐색
