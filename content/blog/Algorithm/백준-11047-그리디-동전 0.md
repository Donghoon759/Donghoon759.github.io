[문제](https://www.acmicpc.net/problem/11047)

```c{3}
#include <cstdio>

int main() {
	int answer = 0;
	int N, K;
	int values[10];
	scanf("%d %d", &N, &K);
	for (int i = 0; i < N; i++) {
		scanf("%d", &values[i]);
	}
	for (int i = N - 1; i > -1; i--) {
		int q = K / values[i];
		if (q >= 1) {
			answer += q;
		}
		K = K % values[i];
		if (K == 0)
			break;
	}
	printf("%d", answer);
}
```

- 모든 상황에서 항상 최선인 경우를 택한다.
