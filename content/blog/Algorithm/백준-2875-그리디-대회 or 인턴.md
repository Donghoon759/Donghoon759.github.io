[문제](https://www.acmicpc.net/problem/2875)

```c++{3}
#include <cstdio>

int main() {
	int N, M, K;
	int girls, mans;
	int teams;
	scanf("%d %d %d", &N, &M, &K);
	teams = N/2 >= M ? M : N/2;
	girls = N - 2 * teams;
	mans = M - teams;
	K -= girls;
	K -= mans;
	while (K>0) {
		teams--;
		K -= 3;
	}
	printf("%d", teams);
}
```

- 팀수는 여자/2와 남자수의 최소
- 남은 여자, 남자 수를 K에서 빼준 후, 더 필요하면 팀 하나를 해체할 때마다 K 3씩 감소
